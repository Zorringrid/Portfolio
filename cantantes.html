Sobre el proyecto

Esta web de cantantes fue desarrollada como práctica del módulo de HTML y CSS en Ironhack. El objetivo era crear una web multipágina dedicada a varios artistas, aplicando conceptos fundamentales como estructura semántica, imágenes, listas, navegación y estilos personalizados.

Descripción general

La web está formada por una página principal con un índice de artistas y tres páginas individuales dedicadas a Justin Bieber, Miley Cyrus y Taylor Swift. Cada página incluye un banner personalizado con la foto del artista, su nombre y enlaces a sus redes sociales. Además, cada artista cuenta con una pequeña biografía, una sección con vídeos de YouTube, y enlaces directos a sus álbumes en Spotify, junto con listas temáticas como discografía, premios o canciones destacadas.

Objetivo académico

  • Practicar estructura semántica en HTML.
  • Aplicar estilos CSS personalizados.
  • Crear listas ordenadas, desordenadas y anidadas.
  • Usar Flexbox para listas horizontales.
  • Diseñar una web multipágina con navegación.
  • Trabajar con imágenes de fondo y formateo de texto.

Funcionalidades principales

Página principal

  • Índice de artistas con enlaces.
  • Lista desordenada.
  • Tarjetas con Flexbox y hover

Páginas individuales

  • Título y descripción del artista.
  • Imagen destacada.
  • Álbumes

Estilos y diseño

  • Imágenes de fondo.
  • Flexbox para listas horizontales.
  • Tipografías y colores personalizados.
  • Diseño responsive básico.
