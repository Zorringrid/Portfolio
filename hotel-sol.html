HOTEL SOL

Sistema de Gestión Hotelera

Proyecto académico desarrollado por el grupo DesArrollados

.NET · SQL Server · XML · Arquitectura por capas · Integración con Odoo

Sobre el proyecto

Aplicación de escritorio desarrollada en C# (.NET / WinForms) como parte del curso académico “Técnicas de persistencia de datos con .NET y programas ERP” de la Universitat Oberta de Catalunya (UOC). Hotel SOL es una aplicación de gestión hotelera diseñada para dar soporte a las operaciones internas de un hotel real: administración de usuarios, gestión de habitaciones, control de reservas, check-in/check-out y solicitud de servicios adicionales. El proyecto fue desarrollado por el equipo DesArrollados como parte de un portafolio profesional. Incluye arquitectura por capas, persistencia de datos mediante ADO.NET, exportación en XML e integración con el ERP Odoo mediante Python.

Descripción general

Hotel SOL simula el funcionamiento interno de un hotel desde el punto de vista de la gestión operativa. Permite administrar usuarios con distintos roles (Administrador, Recepcionista y Cliente), gestionar habitaciones con precios variables según temporada, registrar reservas, realizar check-in/check-out y gestionar servicios adicionales.

objetivo académico

  • Uso de arquitectura en capas.
  • Persistencia de datos con ADO.NET.
  • Separación de responsabilidades mediante DAO.
  • Trabajo con bases de datos SQL Server
  • Integración con un ERP externo (Odoo)-
  • Intercambio de datos mediante XML.

Funcionalidades principales

Sistema de inicio de sesión

  • Validación de credenciales
  • Roles con permisos diferenciados
  • Acceso restringido según tipo de usuario

Gestión de usuarios

  • Creación, edición y eliminación de usuarios
  • Roles: Administrador, Recepcionista, Cliente

Gestión de habitaciones

  • Precios dinámicos según temporada
  • Alta y modificación de habitaciones
  • Control de disponibilidad

Gestión de reservas

  • Creación de reservas por parte del cliente
  • Control de ocupación
  • Registro de servicios adicionales

Operativa del recepcionista

  • Check-in y check-out
  • Consulta de reservas activas
  • Registro de servicios consumidos

Exportación y persistencia

  • Exportación de datos a XML
  • Persistencia completa en SQL Server
  • Carga inicial de datos tras la instalación

Integración con Odoo

  • Envío de datos del hotel hacia Odoo
  • Sincronización de habitaciones y reservas
  • Comunicación mediante Python y XML/API
